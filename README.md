# Coding Performance Analytics & Leaderboard System

## 📌 Overview

This project is a data-driven analytics platform designed to track, analyze, and visualize coding performance of users. It combines problem-level data with user learning behavior to generate meaningful insights, leaderboard rankings, and skill improvement recommendations.

---

## 🚀 Key Features

* 📊 **Performance Tracking:** Monitors coding activity across users including problems solved, difficulty levels, and time trends
* 🏆 **Leaderboard System:** Ranks users based on a composite scoring model using consistency, difficulty, and improvement metrics
* 📈 **Progress Analysis:** Visualizes monthly growth and coding patterns
* 🧠 **Skill Insights:** Identifies weak areas and provides topic-based recommendations
* 📉 **Difficulty Analysis:** Breaks down performance across Easy, Medium, and Hard problems

---

## 🛠️ Tech Stack

* **Python:** Pandas, NumPy (data processing & feature engineering)
* **Data Visualization:** Power BI
* **Data Sources:** Simulated coding activity + LeetCode problem metadata
* **Other Tools:** Excel

---

## ⚙️ Workflow

1. Data preprocessing and cleaning using Pandas
2. Feature engineering (difficulty score, consistency, topic coverage)
3. Synthetic data generation for realistic coding behavior
4. Metric calculation (leaderboard score, ranking, improvement tracking)
5. Dashboard creation in Power BI for interactive visualization

---

## 📊 Dashboard Highlights

* User-wise performance dashboard
* Top performers and leaderboard ranking
* Score distribution and segmentation
* Topic-wise problem-solving analysis
* Difficulty-based performance insights

---

## 💡 Key Insights Generated

* Identification of top-performing users based on multi-factor scoring
* Analysis of coding trends over time
* Detection of weak topics for targeted improvement
* Comparison of users based on performance metrics

## 🎯 Use Case

This system can be used by:
